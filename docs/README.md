## 🚀 快速开始

### 第一次使用改进版本

1. **运行诊断脚本**
   - 打开浏览器控制台
   - 运行 `diagnostic.js`
   - 确认所有检查通过

2. **验证主页配置持久化**
   - 在主页添加测试API端点
   - 切换到任意角色
   - 切换回主页
   - 确认配置恢复

3. **验证角色切换**
   - 为不同角色配置不同的API端点
   - 切换角色
   - 确认配置自动切换

### 遇到问题时

1. **运行诊断脚本**
   - 查看哪些检查失败
   - 记录错误信息

2. **查看控制台日志**
   - 检查是否有JavaScript错误
   - 查找红色错误信息

3. **参考验证清单**
   - 打开 `VERIFICATION_CHECKLIST.md`
   - 按照常见问题排查步骤操作

4. **必要时回滚**
   - 参考 `VERIFICATION_CHECKLIST.md` 的回滚步骤
   - 使用备份文件恢复
